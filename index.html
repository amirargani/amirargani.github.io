Das Problem & Mein Ansatz

Wie ich helfe, Daten produktiv nutzbar zu machen

Das Problem in Unternehmen

Viele Unternehmen erfassen Daten – nutzen sie aber nicht effektiv:

  • Manuelle Prozesse verlangsamen Abläufe.
  • Daten bleiben in Datensilos (z. B. zwischen Zeiterfassung und ERP-Systemen).
  • Fehlende Struktur führt zu schlechter Datenqualität.

Mein Lösungsansatz

Ich entwickle Systeme, die genau diese Probleme lösen:

  • Integration von Daten in bestehende Unternehmensprozesse.
  • Klare Datenstrukturen und saubere Zuordnung.
  • Automatisierung statt manueller Übertragung.
  • Kombination von ERP-Systemen und Machine Learning.

DeepWorkInsights
Python Docker H2O AutoML Auto-Sklearn

Was ist das & Ziel

Ein automatisiertes Datenpipeline- und Vorhersageprojekt, das offizielle monatliche Arbeitsmarktdaten für Deutschland herunterlädt und verarbeitet.

Beschreibung

Downloads official monthly unemployment data from the Federal Employment Agency (BA), parses it into CSV datasets, and runs two independent AutoML engines (H2O AutoML and Auto-sklearn) to forecast the current month's figures. The pipeline is containerized using Docker and Docker Compose.

Verwendete Technologien

Python 3.10+, H2O AutoML, Auto-Sklearn, Docker, Docker Compose, Pandas, NumPy, OpenPyXL
